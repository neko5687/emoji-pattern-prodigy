Rules

You get 15 seconds to solve each pattern. Timer starts after clicking the solve-button. You can only solve once!
You can earn points in the following way:

  • 200 points per difficulty-level (200/🌶️,400/🌶️🌶️,600/🌶️🌶️🌶️)
  • every second passed: -10 points
  • using a hint: -50 points

Try to solve as quick as you can to save the first place in hall of fame.
If you like you can vote on the pattern afterwards.

Good luck!
